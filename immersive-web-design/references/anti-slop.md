# 反AI审美陷阱

## 7大指纹（命中≥3条 → 重新设计）

1. **紫蓝渐变**：几乎所有AI的默认配色。替代：非对称暖冷渐变、单色高饱和强调、加第三色相打破预期
2. **毛玻璃泛滥**：用实色+阴影代替；如果用blur仅限一个焦点元素，叠加噪点纹理
3. **均匀卡片网格**：用不对称布局(80/20)、不用卡片(空间+排版分组)、混合媒介
4. **渐变文字万金油**：仅限一个hero标题；其他用字重差异(900 vs 200)创造层级
5. **居中一切+均匀间距**：左对齐+不对称构图、间距松紧交替、网格断裂
6. **通用字体**：见 typography-palette.md 替代表
7. **bounce/elastic缓动**：用 expo.out / quart.out / quint.out

## 用户要求命中项时

不拒绝需求，通过其他维度突破：排版差异化、空间打破、纹理叠加(噪点/斜纹/半调)、独特动画序列、自定义光标/hover状态。

1. 截图说"AI做的"，对方立刻相信？→ 重新设计
